## 실행

- gradle build
```
gradlew build -x test
```
- docker build
```
docker build --build-arg DEPENDENCY=build/dependency -t [dockerImgname]:[dockerTag]
```
- docker run
```
run  -p [usePort]:8080 [dockerImgname]:[dockerTag]
```
